Street Art Toy

This web app uses the WebRTC getUserMedia API for HTML5 camera access. WebGL fragment shaders (GLSL) are used to apply real-time special effects to the camera video feed. Currently only the following browsers support getUserMedia and WebGL:

  • Google Chrome — enabled in Beta and Canary (Chrome 21 and above), otherwise go to chrome:flags and enable the “MediaStream” flag
  • Opera 12 — go to opera:config and enable WebGL
  • Opera Mobile 12 — it works on Android!
